Being a programmer is more than just write code you need more skills than code writing to be a succesfull programer, team work, patience and resilence are some of the habilities that programmers need in order to face daily challenges

多What is a Program?

A program is a Software entity that allows us to solve a specific problem. We say that it is a Software entity since it is not a set of physical components (such as hardware) but allows us to manipulate them. When we talk about a program, we refer to a set of logical elements (called Algorithms) that are specified, designed and later implemented by the development team.
